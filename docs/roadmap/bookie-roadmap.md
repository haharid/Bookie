# Bookie Roadmap

## Bookie 3.1 — Refactoring

Slogan: Readable Code

Goal:

- Keep all existing features unchanged
- Keep EPUB output identical
- Make the code easier to read and maintain

Included:

- Architecture documentation
- Blankline Cleaner naming / structure cleanup
- Paragraph Cleaner extraction
- General Cleanup

Not included:

- New features
- Regex optimization
- Footnote engine rewrite
- Chapter engine rewrite

## Bookie 3.2 — Foundation

Slogan: Extensible Structure

Goal:

- Build the foundation for safer future feature development

Candidates:

- Template Engine
- CSS Loader
- Debug Mode
- Auto Test
- Logger
- Rule Engine
- Footnote structure review
- Chapter structure review

## Bookie 3.3 — Improvements

Slogan: User-visible Improvements

Goal:

- Add new features and improvements after the foundation is stable

Candidates:

- New EPUB cleanup rules
- New UI tools
- Optional automation
- Performance improvements
- Regex optimization with tests
